🏭 機房檢測輔助工具

🌓

ScheduledPages · 機房檢測助手

定時排程 · Telegram 心跳 · 四種動作類型 — 機房維運自動化

⚡ 4 種動作 · 3 種週期 · 心跳監測

🛠️ 功能一覽

四種動作類型 × 三種週期設定（每日 / 每週 / 一次），涵蓋機房維運所需

🌐

開啟網頁

時間到自動用瀏覽器開啟指定 URL。適用於每日打開 Grafana / Zabbix 儀表板、管理後台。

📝

備忘彈窗

跳出 always-on-top 視窗，顯示自訂內容。支援 {time} {date} 變數。適用於跳出檢測清單逐項確認。

▶️

執行程式

以 subprocess.Popen 執行本機腳本或程式（.bat / .ps1 / .exe）。適用於定時跑 ping 檢測、磁碟檢查。

✈️

Telegram 通知

透過 Bot API 發送訊息到指定 Chat ID（可逗號分隔多人）。支援心跳定時發送與排程觸發。

🌅 每日巡檢流程

自動化早班、午班、下班前的例行工作

07:30

📝 備忘彈窗 — 早班檢測表

跳出完整 checklist：UPS 運作正常、空調溫度 22±2°C、機櫃指示燈、備份狀態、防火牆連線數。

08:00

🌐 自動開啟監控儀表板

瀏覽器自動打開 Grafana / Zabbix 首頁，接班直接檢視。

09:00

▶️ 執行檢測腳本

背景執行 ping 多台核心設備 + 檢查 C 槽剩餘空間，結果寫入日誌。

12:00

✈️ TG 通知 — 午間確認

發送「✅ 機房上午運作正常 {time}」到 Telegram 群組。

17:00

📝 備忘彈窗 — 下班清單

提醒：確認備份完成、檢查異常告警、填寫日報。

📅 每週定期作業

設定每週特定日執行的維護項目

🧹

週一 09:00 — 環境整理

備忘彈窗列出：清潔濾網、檢查線纜標籤、確認備份媒體狀態。

📊

週五 15:00 — 週報彙整

開啟網頁 → 管理後台報表頁面，截圖準備週報。

🔐

週六 02:00 — 離峰維護

執行程式 → 執行 Windows Update 檢測腳本或磁碟重組。

⚡ 臨時 / 不定期項目

一次性排程 + 心跳監測即時告警

❤️

心跳監測 (Heartbeat)

每 N 分鐘發送 Telegram 確認主機存活。未收到通知 → 主機關機或斷線 → 人員需到機房確認。

支援變數：✅ 主機正常 {date} {time}

🔔

一次性排程 (Once)

指定日期觸發一次，適用於：年末檢查、設備汰換日提醒、合約到期通知。

📋

排除日期設定

國定假日、特休日可設定排除，該日不觸發排程。避免放假時收到不必要的通知。
