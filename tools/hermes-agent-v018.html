🔮
Mixture-of-Agents（MoA）
多模型 Ensemble，第一公民等級
新功能

把多個模型組合成命名集合，像選單一模型一樣直接選用。參考模型的推理過程會即時顯示，聚合答案以 Streaming 方式呈現。

  • 支援任意 LLM 混搭（OpenAI / Anthropic / 本地模型等）
  • 每個模型的推理過程對你可見，不是黑盒子
  • Aggregator Streaming 输出，邊推理邊看到結果
  • 適合重要決策：研究報告、股票分析、策略評估
🎯
Completion Contracts
/goal — 工作完成標準
新功能

下 /goal 時定義「完成是什麼樣子」，Agent 會對照這個標準驗證自己的工作，不會「感覺做完了」就停。

  • Agent 自己 Judge 完成度
  • 可綁定具體可測量標準（如：本益比、殖利率）
  • 減少來回校正次數
📈
/learn + /journey
自我改進機制
新功能

把自我改善變成可見、可操控的循環。系統會記住每次執行的優缺點，你可以在下次扭轉。

  • /learn：總結這次執行的經驗教訓
  • /journey：看到整個決策鏈的来龍去脈
  • 改進決策品質，不是只是執行
⚡
Background Subagent Fan-out
並列子代理
核心功能

子代理可以在背景並列展開。適合多路徑研究或分析任務，結果同時回來再整合。

  • 最多 3 個子代理並列
  • 各自獨立執行，不需等待彼此
  • 應用：法人報告 + 社群情緒 + Vault 歷史三路並列
💻 桌面功能 3 功能
📁
Desktop Projects
一級編碼專案支援
新功能

將工作區綁定到真實資料夾（如 ~/Documents/StockResearch/），所有工具鏈、Cline/VSCode 直接作用在該目錄。

  • session 的 workspace 直接在指定目錄
  • 綁定後 tools 自動具備該目錄上下文
  • 適合長期研究專案（股票分析、內容企劃）
🧠
Memory Graph
記憶圖譜視覺化
新功能

將 vault 中散落的筆記找出關聯並視覺化。Obsidian 的 contacts / Spaces / DailyNote 都能串成可探索的節點圖。

  • 探索 vault 內的隱藏關聯
  • 人物、空間、筆記的連結一覽
  • 發現以前沒注意到的關係
🖥️
電腦使用（CUA Driver）
直接操控桌面 GUI
既有用

CUA（Computer Use Agent）模式：點擊、輸入、截圖，全自動驅動本機桌面應用（Chrome、Obsidian 等）。

  • 可用於自動化網頁操作
  • 驅動 Obsidian 直接寫筆記
  • 需 macOS/Windows/Linux 桌面版
⚙️ 系統 / 安全 / 部署 5 功能
🔒
MCP 安全加固
設定持久化強化
安全

MCP 設定持久化的攻擊面已全面硬化，block base_url 滲漏，並為短 TTL session 加入 keepalive。

  • 封鎖 base_url exfiltration
  • Session keepalive 防止中斷
  • 攻擊面大幅縮小
📦
Gateway Scale-to-Zero
生產級部署
基礎設施

Gateway 現在可以真正 Scale-to-Zero，閒置時完全釋放資源，並具備 drain 協調機制。

  • 零實例時不收費
  • Drain 協調確保請求不丢失
  • 支援 Kubernetes / 雲函數部署
🎵
AAC 語音備忘錄 Remux
Signal 格式強化
既有強化

Signal 訊號格式升級：支援 AAC 語音備忘錄自動 Remux，並統一各平台的 Markdown 格式化。

  • AAC 語音檔自動處理
  • Markdown 跨平台一致
  • 改善 Telegram / Signal 轉發忠誠度
☁️
多平台 Adapter 統一捆綁
一鍵開通
基礎設施

Slack / DingTalk / WhatsApp / Matrix / Feishu / Telegram / WeCom / Email / SMS 全部遷移到統一捆綁架構。

  • 一個安裝包含所有平台
  • 設定更簡單，維護更容易
  • 減少重複適配代碼
🗄️
Blank Slate Setup Mode
極簡起步
既有強化

新用戶預設為最小化 Agent，按需逐步開通功能（而非一開始全部轟炸）。

  • 極簡起步，降低學習曲線
  • 漸進式開通功能
  • 減少第一次上手的認知負擔
